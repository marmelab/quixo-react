# quixo-react

Quixo Game Using NodeJS & react-native !

## Install

Install docker and Expo for the react native app

```
make install
```

## Start

Start the game

```
make start
```

## Test

Runs all the tests of the project

```
make test
```

## Help

List all commands

```
make help
```
